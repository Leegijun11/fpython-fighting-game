# 🥊 Fpython 격투 게임

파이썬으로 만든 텍스트 기반 RPG 격투 게임입니다.  
캐릭터를 선택하고 가위바위보로 빌런들과 전투하며 전리품을 획득해 능력치를 키울 수 있습니다.

---

## 🎮 게임 소개

개발자 세계의 빌런들(오타몬, 기면증몬, 릴스몬 등)과 **가위바위보** 방식으로 1:1 결투를 벌입니다.  
승리하면 랜덤 전리품을 획득하고 영구적으로 능력치가 상승합니다.

---

## ⚙️ 설치 및 실행

**Python 3.x** 이상이 필요합니다. 별도 라이브러리 설치는 필요하지 않습니다.

```bash
# 레포지토리 클론
git clone https://github.com/your-username/fpython-fighting-game.git
cd fpython-fighting-game

# 게임 실행
python Update.py
```

---

## 📖 게임 방법

1. 게임 실행 후 **캐릭터를 선택**합니다.
2. **전투 시작**을 선택하면 랜덤 적이 등장합니다.
3. **가위바위보**로 승부합니다. (초성 입력도 지원)
   - `ㄱㅇ` 또는 `가위`
   - `ㅂㅇ` 또는 `바위`
   - `ㅂ` 또는 `보`
4. 적을 쓰러뜨리면 **랜덤 전리품**을 획득하고 능력치가 오릅니다.
5. **내 정보 보기**에서 현재 스탯과 승률을 확인할 수 있습니다.
6. 종료 시 **저장(save.txt)** 하거나 캐릭터를 **은퇴**시킬 수 있습니다.
